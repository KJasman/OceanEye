# OceanEye: Marine Detection
### SOLIDS Lab, Uvic
#### In collaboration with SPECTRAL Lab, UVic

## Installation
 * Download the __setup.bat__ file
 * Locate and run __setup.bat__  
 * Run the app:
 ```streamlit run app.py```
 * A browser page will open showing the running app
